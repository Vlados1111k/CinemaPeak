# Звіт за ітерацію №1 (Лабораторна 34)

## Що зроблено:
1. **Архітектура** Створено Solution з 5 проєктами (Domain, Application, Infrastructure, Console, Tests) згідно з принципами Clean Architecture.
2. **Модель предметної області:** Реалізовано базові класи для системи CinemaPeak: `Movie`, `Ticket` (з ієрархією `StandardTicket` та `VipTicket`), `Session`.
3. **Вертикальний зріз:** Реалізовано повний шлях від консольного введення через сервіс бронювання до збереження в `InMemoryRepository`.
4. **Якість коду:** - Додано інкапсуляцію та перевірку інваріантів у конструкторах.
   - Написано 5 юніт-тестів для перевірки логіки домену.
5. **CI/CD:** Налаштовано GitHub Actions, який автоматично збирає проєкт та запускає тести при кожному пуші.
